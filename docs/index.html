Documentation

This program is designed for D&D 5e, with the intent of easing the process of handing out treasure found on enemies, separate from treasure hoards.

Running the program without any arguments will produce a Graphical User Interface. It is simple to use. Simply input the amount of enemies of the different levels, and press roll

Entering an invalid value will set that field to zero.

If you run the program with "-nogui" as the first argument, the program will start with a command line user interface. This is fully functional, but inexperienced users may prefer the GUI.

Alternatively, you can run the program with an arbitrary number of argument pairs. These pairs must be numbers, and must be formatted as "< level > < amount >"
